# CI Regress Kit

Use this kit to detect contract and behavior regressions quickly.

## Recommended Commands

- `make lint-fast`
- `make test-fast`
- `make test-contracts`
- `make test-conformance`
- `make test-acceptance`
- `make test-adoption`
- `make test-uat-local`
- `make test-runtime-slo`
- `make test-scale-profile`
- `make test-serve-slo`
- `make coverage`
- `make sast-fast`
- `make license-check`

## Required Artifact Capture

- `./wrkr-out/reports/*`
- `./wrkr-out/jobpacks/*`
